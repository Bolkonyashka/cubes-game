*****ОПИСАНИЕ*****

Данное приложение является реализацией игры "Кубики".

*****ПРАВИЛА И МЕХАНИКА ИГРЫ*****
- Основная цель игры - заработать как можно больше очков 
- Количество очков отображается в поле "Счет"
- Заработать очки можно, щёлкая мышью по группам кубиков одного цвета 
- Группа кубиков считается таковой, если состоит как минимум из двух кубиков 
- Чем больше кубиков в группе, тем больше очков начисляется 
- По ходу игры можно узнавать потенциально возможное количество очков за уничтожение группы 
 кубиков простым наведением мыши на оную группу, 
 при этом сама группа будет выделена другим оттенком цвета для упрощения зрительного восприятия 
- Потенциальные очки отображаются в поле "Возможные очки"
- В любой момент игры возможны сохранение или загрузка игры
- Для корректного сохранения и загрузки ваш никнейм должен состоять из десяти символов (только буквы и цифры)
- Загрузка игры возможна только если под вашим никнеймом уже существует сохраненная игра
- Когда не останется ни одной группы кубиков, которую можно уничтожить, 
 игра автоматически закончится, при этом дав возможность сохранить свой результат нажатием кнопки "Сохранить"
- Список лидеров находится в правом нижнем углу окна игры
- Перезапуск игры возможен в любое время по ходу игры, посредством нажития кнопки "Рестарт", 
 находящейся в правой части игрового окна

*****ПОДРОБНОСТИ РЕАЛИЗАЦИИ*****

- За состояние логики игры отвечает класс Game.
- Текущее состояние игрового пространства хранится в поле Game.map . 
- Метод Game.game_starter задает начальные параметры для запуска и отображения игры и вызывает
 отрисовку игрового окна.
- Методы Game.(mouse_down/blacklight/same_cubes_search/shift_columns_search/map_refresh и др.) отвечают 
 за механику игры, реагирование на нажатие на блок кубиков и тд.
- Методы Game.(load_game/save_game/restarter) отвечают за загрузку, сохранение и перезапуск игры.
- Методы с названием Game.xxx_painter отвечают за отрисовку игрового окна и его составляющих.
